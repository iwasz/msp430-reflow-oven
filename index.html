===What it is=== Firmware for my simple reflow-owen controller.

===Build===

git clone https://github.com/iwasz/msp430-reflow-owen.git
cd msp430-reflow-owen/build
# Edit configure.sh so it points to MSP-ware
./configure.sh
make 
make upload
